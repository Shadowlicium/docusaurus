# Schéma

![](/img/appli.png)

Ce schéma montre comment l'utilisateur et l'administrateur se connectent à Intune.  
L'administrateur se connecte sur Intune via la sécurité multiple avec Microsoft Authenticator, qui demandera un code ainsi qu’un code PIN, une empreinte ou une reconnaissance faciale pour valider ensuite la connexion.  
L'utilisateur aura exactement le même type de connexion, mais uniquement pour pouvoir utiliser les outils de travail ; cela n'impactera en aucun cas son utilisation personnelle.
